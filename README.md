# Promplr

**Store, version, and share AI prompts.**

> GitHub for Prompts — with one-click copy, embeddable widgets, and community discovery.

## Features (Coming Soon)

- 📝 Store & organize prompts with categories
- 🔄 Version history & diff view
- 🔗 Share public or keep private
- 📋 One-click copy
- 🧩 Embeddable widgets for Notion/Medium/Substack
- 👤 Public profile pages
- 🔐 Auth: Email, Google, GitHub

## Tech Stack

- **Frontend**: Next.js 14, TypeScript, Tailwind CSS, shadcn/ui
- **Backend**: Next.js API Routes, Prisma ORM
- **Database**: PostgreSQL (Supabase)
- **Auth**: NextAuth.js
- **Hosting**: Vercel

## Development

```bash
pnpm install
pnpm dev
```
